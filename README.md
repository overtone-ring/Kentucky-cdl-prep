# Kentucky CDL Practice Test

A mobile-friendly web application for practicing the Kentucky Commercial Driver's License (CDL) exam.

## Features

- **4 Test Categories**: General Knowledge (50 questions), Air Brakes (25), Passenger (20), School Bus (20)
- **Practice Mode**: Get immediate feedback after each question with explanations
- **Test Mode**: Simulate the real test with no feedback until the end
- **Progress Tracking**: Your scores are saved locally in your browser
- **Dark Mode**: Automatically adapts to your device's theme
- **Mobile-First**: Designed for easy use on smartphones
- **Offline Ready**: Works without an internet connection once loaded

## Usage

Visit the live site: [Kentucky CDL Prep](https://overtone-ring.github.io/Kentucky-cdl-prep/)

Or download `index.html` and open it directly in your browser.

## Based On

Questions are derived from the official Kentucky CDL Driver Manual.
