# Kentucky CDL Practice Test

A mobile-friendly web application for practicing the Kentucky Commercial Driver's License (CDL) exam. All 230 questions have been verified against the official Kentucky CDL Driver Manual.

## Test Categories

| Category | Questions | Required For |
|----------|-----------|--------------|
| General Knowledge | 50 | All CDL classes |
| Air Brakes | 25 | Vehicles with air brakes |
| Pre-Trip Inspection | 30 | Skills test preparation |
| Combination Vehicles | 25 | Class A CDL |
| Doubles/Triples | 20 | T endorsement |
| Tank Vehicles | 15 | N endorsement |
| Hazardous Materials | 25 | H endorsement |
| Passenger | 20 | P endorsement |
| School Bus | 20 | S endorsement |

## How to Use

### Getting Started
1. Visit the live site: [Kentucky CDL Prep](https://overtone-ring.github.io/Kentucky-cdl-prep/)
2. Or download `index.html` and open it directly in your browser

### Practice Mode vs Test Mode
- **Practice Mode** (toggle ON): Get immediate feedback after each question with explanations. Great for learning.
- **Test Mode** (toggle OFF): No feedback until you finish. Simulates the real CDL test experience.

### Taking a Test
1. Select the test category you want to practice
2. Choose Practice or Test mode
3. Answer questions using the letter buttons (A, B, C, D) or keyboard shortcuts (1-4 or A-D)
4. Use arrow keys or Next/Previous buttons to navigate
5. Review your results and explanations when finished

### Progress Tracking
Your scores are automatically saved in your browser. View your best and most recent scores on the home screen.

## Features

- **Verified Questions**: All questions cross-referenced with the official KY CDL Manual
- **Instant Feedback**: Learn from explanations after each answer in Practice Mode
- **Score Tracking**: Your progress saves locally in your browser
- **Dark Mode**: Automatically matches your device theme
- **Mobile-First**: Works great on phones, tablets, and desktops
- **Offline Ready**: Works without internet once loaded
- **Keyboard Shortcuts**: Quick answer selection with number or letter keys

## For School Bus Drivers (Class B with P and S)

If you're getting your School Bus endorsement, focus on these tests:
1. General Knowledge (required for all)
2. Air Brakes (if your bus has air brakes)
3. Passenger (P endorsement)
4. School Bus (S endorsement)
5. Pre-Trip Inspection (for skills test)
